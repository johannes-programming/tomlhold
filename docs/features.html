Features of tomlhold

The Holder class

Holder is a dict-like class specifically for handling toml. It implements all operations and methods of dict.

class tomlhold.Holder(data: Any = None)

Create a new Holder object.

data

The data property of Holder represents the entire instance. Its getter returns a dict. Its setter interprets the given value to change the instance accordingly.
