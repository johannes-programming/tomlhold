Features of tomlhold

The Holder class

Holder is a dict-like class specifically for handling toml. It implements all operations and methods of dict.

class v440.core.Local.Local(data: Any = None)

Create a new Local object. The one (non required) positional argument is passed to its data property.

data

The data property of Local represents the entire instance. Its getter returns a list. Its setter interprets the given value as a local version identifier and uses this to change the instance accordingly.
